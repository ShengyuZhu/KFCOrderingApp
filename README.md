# KFCOrderingApp
# 肯德基点餐系统

## 项目简介
这是一个基于Java的控制台应用程序，模拟了肯德基点餐的基本流程。用户可以通过命令行界面查看菜单、添加菜品到购物车、查看购物车内容、进行结算以及退出系统，实现了简单的点餐交互功能。


## 功能说明
1. **查看菜单**：显示肯德基的菜品列表及对应价格
2. **添加菜品到购物车**：选择菜品编号和数量，将菜品加入购物车（若菜品已在购物车中，会自动累加数量）
3. **查看购物车**：显示当前购物车中的所有菜品、数量及总价
4. **结算**：完成订单结算并清空购物车
5. **退出**：结束程序运行


## 使用方法
1. 确保本地已安装Java运行环境（JDK 8及以上版本）
2. 将代码保存为`KFCOrderingApp.java`文件
3. 编译代码：在命令行中执行`javac KFCOrderingApp.java`
4. 运行程序：执行`java KFCOrderingApp`
5. 根据控制台提示进行操作：
   - 输入`1`查看菜单
   - 输入`2`添加菜品（需先选择菜品编号，再输入数量）
   - 输入`3`查看购物车
   - 输入`4`进行结算
   - 输入`5`退出系统


## 代码结构
- **MenuItem类**：存储菜品信息（名称、价格），提供获取菜品信息的方法
- **CartItem类**：表示购物车中的菜品项，包含对应的菜品、数量及计算单项总价的方法
- **KFCOrderingSystem类**：点餐系统核心类，实现菜单初始化、菜单展示、购物车管理（添加、查看、结算）等功能
- **KFCOrderingApp类**：程序入口类，通过控制台交互接收用户输入并调用系统功能


## 注意事项
- 输入菜品编号时，请确保在菜单显示的有效编号范围内
- 输入数量时，请输入正整数
- 结算后购物车会自动清空，如需继续点餐可重新添加菜品
